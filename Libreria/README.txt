
Sistema de Gestión de Biblioteca — Prototipo (Estructuras de Datos Lineales)
===========================================================================
Cómo ejecutar
-------------
1) Requiere Python 3.10+
2) Ejecuta:
   python /mnt/data/library_system.py
   (o descarga el archivo y ejecútalo localmente: python library_system.py)

Qué demuestra
--------------
- Lista (list): catálogo de libros, usuarios, historial.
- Cola (deque): lista de espera por libro (FIFO).
- Pila (list): deshacer última operación (LIFO).
- Arreglo (list con índices): generación de IDs secuenciales.

Operaciones
-----------
- Registrar usuario
- Buscar/Listar libros
- Préstamo y devolución con lista de espera
- Deshacer última operación (undo)
